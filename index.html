Connect Gmail

Enter your Gmail Client ID to enable sending. You'll need to create a Google Cloud project with the Gmail API enabled.

  • Go to console.cloud.google.com → New project
  • Enable Gmail API → Create OAuth credentials
  • Add your domain/localhost as authorised origin
  • Copy the Client ID and paste below

Configure SMS via Twilio

Twilio sends real SMS messages. Create a free account at twilio.com, buy a number (~$1/mo), and paste your credentials below.

  • Sign up at twilio.com → get Account SID + Auth Token
  • Buy a phone number to send from
  • Paste all three below — stored locally only
SMS sends through a local proxy because browsers block direct Twilio calls. See the README below for the one-command proxy setup.
